Напомним, что палиндром — это строка, которая одинаково читается как слева направо, так и справа налево. Вам дан набор различных строк. Найдите среди них такие пары, которые при конкатенации дают палиндром. Более формально, найдите все пары (i, j), что i != j и si+sj является палиндромом.

Формат ввода
В первой строке дано единственное число — количество строк n (1 <= n <= 10^5). В каждой из следующих n строк заданы строки si по одной на строке. Сумма длин всех строк не превосходит 106. Строки состоят из маленьких латинских букв, строки не бывают пустыми.

Формат вывода
Выведите все найденные пары индексов (нумерация с единицы), упорядоченные по первому индексу, а при равенстве первых — по второму.

Пример 1
Ввод	Вывод
4
a
abbaa
bba
abb
        1 2
        1 3
        2 3
        3 4
        4 1
        4 3
Пример 2
Ввод	Вывод
4
pa
lap
palk
pal
        1 2
        2 4
        3 2
        4 2
Пример 3
Ввод	Вывод
2
aba
abba

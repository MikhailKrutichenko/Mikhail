Disclaimer: задача повышенной сложности. Для её решения необходимо самостоятельно найти информацию и разобраться в алгоритме поиска образцов в тексте. Вам может пригодиться алгоритм Ахо-Корасик.

Реализуйте алгоритм поиска набора строк в большом тексте. Вам дана строка t — это текст. Также дан набор из n строчек — это образцы si. Для каждого образца надо найти все его вхождения в текст t.

Формат ввода
В первой строке дан текст t — это строка, состоящая из маленьких латинских букв и не превышающая 10^6 в длину. В следующей строке дано число n (1 <= n <= 10^4). В каждой из следующих n строк даны образцы по одному в строке. Образцы состоят из маленьких латинских букв. Сумма длин всех образцов не превосходит 106.

Формат вывода
Для каждого образца в том порядке, в котором они даны во входных данных, выведите сначала сам образец, а затем на той же строке все позиции вхождений его в текст. Позиции в тексте нумеруются с единицы. Позиции для одного образца надо выводить в отсортированном по возрастанию порядке.
